## 一、核心骰子系统

### 1.1 D20 骰子 —— 一切检定的基础

博德之门3的所有检定都基于 **D20**（20面骰子）。当你进行任何需要判定成功或失败的行为时，系统会投掷一个D20，然后加上各种修正值，与目标数值比较。

**通用检定公式：**
```
检定结果 = D20 + 属性调整值 + 熟练加值（如果有） + 其他修正
```

### 1.2 常见骰子表示法

| 表示法 | 含义 | 示例 |
|--------|------|------|
| 1d20 | 投掷1个20面骰 | 攻击检定、技能检定 |
| 2d6 | 投掷2个6面骰，求和 | 武器伤害 |
| 1d8 | 投掷1个8面骰 | 法术伤害 |
| 1d10 | 投掷1个10面骰 | 长剑伤害 |
| 1d12 | 投掷1个12面骰 | 巨斧伤害 |

### 1.3 三种核心检定类型

游戏中所有概率系统分为三大类：

1. **攻击检定** —— 决定攻击是否命中
2. **豁免检定** —— 决定是否能抵抗法术/效果
3. **技能检定** —— 决定探索/交互行为是否成功

## 二、六大属性系统

### 2.1 属性与调整值

每个角色有六项基本属性，数值范围通常为 8-20。属性值决定**调整值**，调整值才是实际参与计算的数值。

**调整值计算公式：**
```
调整值 = (属性值 - 10) / 2，向下取整
```

| 属性值 | 调整值 | 属性值 | 调整值 |
|--------|--------|--------|--------|
| 8-9 | -1 | 14-15 | +2 |
| 10-11 | 0 | 16-17 | +3 |
| 12-13 | +1 | 18-19 | +4 |
| 20 | +5 | | |

### 2.2 六大属性详解

| 属性 | 英文 | 影响内容 | 关键职业 |
|------|------|----------|----------|
| **力量** | Strength | 近战攻击/伤害、负重、 athletics 技能 | 战士、野蛮人、圣武士 |
| **敏捷** | Dexterity | 远程攻击/伤害、AC（护甲值）、先攻、 stealth/acrobatics | 游荡者、游侠、武僧 |
| **体质** | Constitution | 生命值、体质豁免（维持专注） | 所有职业 |
| **智力** | Intelligence | 法师法术、调查、奥秘、历史 | 法师、诡术师 |
| **感知** | Wisdom | 牧师/德鲁伊/游侠法术、察觉、洞悉、生存 | 牧师、德鲁伊、游侠 |
| **魅力** | Charisma | 吟游诗人/圣武士/术士/邪术师法术、说服、欺骗、威吓 | 吟游诗人、圣武士、术士、邪术师 |

### 2.3 属性分配建议

- **主属性**：你的职业核心属性，尽量点到 16-17（创建角色时）
- **体质**：建议至少 14，影响血量和专注豁免
- **敏捷**：即使非敏捷职业，也建议不低于 10，影响先攻和AC

## 2.5 熟练项系统（Proficiency）

熟练项是 D&D 5E 最核心的角色成长机制之一。它代表你的角色在某些领域受过专业训练，能够比普通人表现得更好。

### 2.5.1 熟练加值（Proficiency Bonus）

**熟练加值是一个只与角色总等级挂钩的固定数值**，与你的属性、职业、种族无关。

| 角色总等级 | 熟练加值 (PB) |
|-----------|--------------|
| 1-4 级 | +2 |
| 5-8 级 | +3 |
| 9-12 级 | +4 |
| 13-16 级 | +5 |
| 17-20 级 | +6 |

> 💡 **博德之门3中角色最高12级**，所以游戏中熟练加值范围是 **+2 到 +4**。

### 2.5.2 熟练项的五大类别

#### ① 武器熟练（Weapon Proficiency）

决定你能熟练使用哪些武器。**不熟练的武器依然可以使用，但会有严重惩罚！**

| 武器类型 | 说明 | 典型职业 |
|---------|------|---------|
| **简易武器** | 匕首、短棒、轻型十字弓、投石索等基础武器 | 几乎所有职业 |
| **军用武器** | 长剑、巨剑、长弓、战锤等高级武器 | 战士、圣武士、野蛮人、游侠 |

**⚠️ 不熟练武器的严重惩罚：**
- 使用不熟练的武器进行攻击时，**攻击检定具有劣势**（投2个D20取低）
- 这意味着你的命中率大幅下降，非常不推荐

**示例：**
- 法师没有长剑熟练项，强行装备长剑 → 每次攻击都有劣势
- 战士拥有所有武器熟练项，可以随意使用任何武器

#### ② 护甲熟练（Armor Proficiency）

决定你能熟练穿着哪些护甲。**不熟练的护甲惩罚比武器更严重！**

| 护甲类型 | 说明 | 典型职业 |
|---------|------|---------|
| **轻甲** | 皮甲、镶钉皮甲、兽皮甲 | 游荡者、游侠、邪术师、吟游诗人、武僧 |
| **中甲** |  hide甲、链甲衫、鳞甲、胸甲 | 德鲁伊、牧师、战士、圣武士、游侠 |
| **重甲** | 链甲、板条甲、板甲 | 战士、圣武士、牧师（部分领域） |
| **盾牌** | 木盾、钢盾 | 战士、圣武士、牧师、德鲁伊 |

**⚠️ 不熟练护甲的毁灭性惩罚：**
- **所有属性检定具有劣势**（包括技能检定如察觉、游说等）
- **所有豁免检定具有劣势**
- **所有以力量或敏捷进行的攻击检定具有劣势**
- **无法在该护甲下施展法术**

> 🚨 **结论：绝对不要穿不熟练的护甲！** 这比不熟练武器严重得多，会让你的角色全面瘫痪。

**示例：**
- 游荡者只有轻甲熟练，强行穿中甲 → 所有检定、豁免、攻击都有劣势，且不能施法
- 法师没有护甲熟练，只能穿普通衣物或使用"法师护甲"法术

#### ③ 技能熟练（Skill Proficiency）

决定你在哪些技能上能加上熟练加值。

**获取方式：**
- **职业**：每个职业提供2-4个自选技能熟练项
- **背景**：每个背景提供2个固定技能熟练项
- **种族**：部分种族提供特定技能熟练项（如半精灵的2个自选技能）
- **专长**：如"技能专家"专长可增加技能熟练项

**效果：**
```
熟练技能检定 = D20 + 属性调整值 + 熟练加值 + 其他修正
非熟练技能检定 = D20 + 属性调整值 + 其他修正（少了熟练加值！）
```

> 💡 在博德之门3中，**非熟练技能没有额外惩罚**（不像武器和护甲），只是少了熟练加值。所以任何人都可以尝试任何技能，只是熟练的人成功率更高。

**游荡者的"专精"（Expertise）：**
- 游荡者可以将**已熟练**的2个技能的熟练加值**翻倍**
- 例如：5级游荡者，察觉技能有专精 → 察觉检定 = D20 + 感知调整值 + **+6**（+3×2）
- 这是游戏中检定加值最高的手段之一

#### ④ 豁免熟练（Saving Throw Proficiency）

决定你在哪些属性的豁免检定中能加上熟练加值。

**每个职业在1级时固定获得2种豁免熟练：**

| 职业 | 豁免熟练 | 说明 |
|------|---------|------|
| 野蛮人 | 力量、体质 | 抗推搡、抗毒素 |
| 吟游诗人 | 敏捷、魅力 | 抗陷阱、抗魅惑 |
| 牧师 | 感知、魅力 | 抗心灵控制、抗恐惧 |
| 德鲁伊 | 智力、感知 | 抗法术、抗魅惑 |
| 战士 | 力量、体质 | 最硬的前排 |
| 武僧 | 力量、敏捷 | 全面防御 |
| 圣武士 | 智慧、魅力 | 抗魅惑、抗恐惧 |
| 游侠 | 力量、敏捷 | 野外生存 |
| 游荡者 | 敏捷、智力 | 躲陷阱、抗心灵 |
| 术士 | 体质、魅力 | 维持专注、抗魅惑 |
| 邪术师 | 感知、魅力 | 抗心灵、抗恐惧 |
| 法师 | 智力、感知 | 抗法术、抗魅惑 |

**效果：**
```
熟练豁免 = D20 + 属性调整值 + 熟练加值
非熟练豁免 = D20 + 属性调整值
```

> 💡 豁免熟练非常重要！例如法师不熟练体质豁免，被击中后维持专注法术时很容易失败；而术士有体质豁免熟练，维持专注能力强得多。

#### ⑤ 工具熟练（Tool Proficiency）

在博德之门3中存在感较低，但在桌面D&D中用于特定工匠活动。

| 工具类型 | 用途 |
|---------|------|
| 盗贼工具 | 开锁、解除陷阱（游荡者必备） |
| 草药工具 | 制作药水、识别草药 |
| 毒药工具 | 制作毒药 |
| 易容工具 | 伪装身份 |
| 伪造工具 | 伪造文件 |
| 各种乐器 | 吟游诗人表演 |

> 💡 在博德之门3中，**盗贼工具**是最实用的工具熟练。没有盗贼工具熟练的角色也可以尝试开锁，但熟练的角色成功率更高。

### 2.5.3 熟练加值的重要规则

**① 同一次掷骰只能加一次熟练加值**

即使有两条不同规则允许你加熟练加值，你也只能加一次。

**示例：**
- 某特性让你对感知豁免加熟练加值
- 你本身就有感知豁免熟练
- 结果：还是只加一次熟练加值，不会叠加成两倍

**② 熟练加值可以被加倍或减半**

某些特性会让你在特定检定中**加倍**熟练加值（如游荡者的专精）。

```
专精技能检定 = D20 + 属性调整值 + (熟练加值 × 2)
```

**③ 没有熟练项时，熟练加值为0**

如果你进行某项检定时没有对应的熟练项，即使你有加倍熟练加值的特性，也不能加任何熟练加值。

```
没有熟练项 + 加倍特性 = 0 × 2 = 0（不加！）
```

**④ 兼职时熟练加值按总等级计算**

```
5级战士 + 3级法师 = 8级总等级 → 熟练加值 +3
```

但**豁免熟练和技能熟练只在1级时获得**，兼职不会额外获得新的熟练项。

### 2.5.4 如何查看自己的熟练项

在博德之门3中：
1. 打开角色面板（按C键）
2. 查看"熟练项"标签页
3. 绿色标记 = 熟练，灰色 = 不熟练
4. 武器和护甲的熟练度会显示在装备界面的图标上

### 2.5.5 熟练项获取途径总结

| 来源 | 获得内容 |
|------|---------|
| **职业** | 武器熟练、护甲熟练、豁免熟练（2种）、技能熟练（2-4个自选） |
| **背景** | 技能熟练（2个固定）、工具/语言熟练 |
| **种族** | 武器熟练（部分种族）、技能熟练（部分种族）、语言 |
| **子职业** | 额外武器/护甲熟练（部分子职业） |
| **专长** | 额外技能/工具/语言熟练 |
| **装备** | 某些装备提供临时熟练（如"知识卷轴"） |

### 2.5.6 新手常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 法师穿中甲 | 所有检定劣势，不能施法 | 法师只能穿布甲或使用"法师护甲" |
| 游荡者装备巨剑 | 攻击有劣势 | 选择轻武器或灵巧武器 |
| 忽视豁免熟练 | 容易被控、法术被打断 | 了解你的职业豁免熟练，优先提升对应属性 |
| 全队没有盗贼工具熟练 | 很多宝箱和门打不开 | 至少让一个队友学盗贼工具 |
| 兼职只为护甲熟练 | 浪费等级，属性不达标 | 兼职前先确认属性要求 |

## 三、战斗系统详解

### 3.1 回合制战斗基础

博德之门3采用**回合制**战斗，进入战斗后每个角色按**先攻顺序**轮流行动。

**每回合每个角色拥有：**
- **1个动作 (Action)** —— 攻击、施法、使用物品等
- **1个额外动作 (Bonus Action)** —— 喝药水、某些特殊能力
- **移动 (Movement)** —— 根据速度移动一定距离
- **1个反应 (Reaction)** —— 借机攻击、某些法术/能力

### 3.2 攻击检定 —— 命中计算

当你发起攻击时，系统会进行一次攻击检定：

```
攻击检定 = D20 + 攻击调整值

如果 攻击检定 ≥ 目标AC → 命中！
如果 攻击检定 < 目标AC → 未命中
```

**攻击调整值计算：**

| 攻击类型 | 调整值 |
|----------|--------|
| 一般近战武器 | 熟练加值 + 力量调整值 + 武器附魔 |
| 灵巧近战武器 | 熟练加值 + 力量/敏捷（取高）+ 武器附魔 |
| 远程武器 | 熟练加值 + 敏捷调整值 + 武器附魔 |
| 法术攻击 | 法术熟练加值 + 对应属性调整值 |

**法术对应属性（施法关键属性 / Spellcasting Ability）：**

| 职业 | 施法关键属性 | 说明 |
|------|-------------|------|
| **法师 (Wizard)** | **智力** | 法师通过学习奥术知识施法，智力越高法术越强 |
| **奥秘骑士 (Eldritch Knight)** | **智力** | 战士子职业，将奥术融入战斗 |
| **诡术师 (Arcane Trickster)** | **智力** | 游荡者子职业，用魔法辅助潜行和偷袭 |
| **牧师 (Cleric)** | **感知** | 通过信仰获得神力 |
| **德鲁伊 (Druid)** | **感知** | 与自然沟通获得法术 |
| **游侠 (Ranger)** | **感知** | 半施法者，感知决定法术效果 |
| **吟游诗人 (Bard)** | **魅力** | 用音乐和表演施展魔法 |
| **圣武士 (Paladin)** | **魅力** | 凭借坚定的信念和誓言施法 |
| **术士 (Sorcerer)** | **魅力** | 天生具有魔法血脉 |
| **邪术师 (Warlock)** | **魅力** | 与异界存在签订契约获得力量 |

> 💡 **为什么法师用智力？** 在D&D设定中，法师的法术来自于**学习和研究**——阅读法术书、理解奥术原理、记忆复杂的咒语手势。智力代表学习能力和逻辑思维，所以智力越高，法师对法术的掌握就越深，法术就越难被抵抗。

> 💡 **为什么牧师用感知？** 牧师的法术来自神明的赐予，感知代表**直觉、洞察力和与超自然力量的连接能力**。感知越高，牧师与神明的联系越强，神术效果越好。

> 💡 **为什么术士/邪术师/吟游诗人/圣武士用魅力？** 这些职业的法术力量来自于**内在天赋、个人意志、与异界存在的交涉能力或感召力**。魅力代表个人气场和影响力，所以是他们的施法关键属性。

### 🔑 施法关键属性的三大核心作用

施法关键属性（Spellcasting Ability）不只是"设定 flavor"，它直接决定了你所有法术的实际战斗效果。

#### ① 决定法术攻击调整值（Spell Attack Modifier）

当你使用**攻击掷骰型法术**（如火球术、魔法飞弹等需要投骰命中的法术）时：

```
法术攻击检定 = D20 + 熟练加值 + 施法关键属性调整值
```

**示例：** 5级法师，智力18（调整值+4），熟练加值+3
- 法术攻击检定 = D20 + 3 + 4 = D20 + 7
- 对AC 15的敌人，需要投出8或更高才能命中（成功率65%）

**如果智力只有14（调整值+2）：**
- 法术攻击检定 = D20 + 3 + 2 = D20 + 5
- 对AC 15的敌人，需要投出10+（成功率55%）

> ⚠️ **结论**：施法关键属性每差2点，命中率就差5%！

#### ② 决定法术豁免DC（Spell Save DC）

当你使用**豁免检定型法术**（如魅惑人类、定身术、蛛网术等让敌人投骰抵抗的法术）时：

```
法术豁免DC = 8 + 熟练加值 + 施法关键属性调整值
```

**示例：** 同上5级法师
- 法术豁免DC = 8 + 3 + 4 = 15
- 敌人需要进行豁免检定，D20+调整值 ≥ 15 才能抵抗成功

**DC数值的意义：**

| 法术豁免DC | 敌人抵抗难度 | 典型场景 |
|-----------|-------------|---------|
| 13 | 较容易抵抗 | 施法关键属性14的低级施法者 |
| 15 | 中等难度 | 施法关键属性18的中级施法者 |
| 17 | 很难抵抗 | 施法关键属性20的高级施法者 |
| 19+ | 极难抵抗 | 高等级+高属性+装备加成 |

> 💡 **DC每增加1点**，敌人抵抗成功率平均下降5%。所以施法关键属性从18提升到20（DC 15→16），相当于给所有敌人施加了5%的劣势！

#### ③ 决定可准备法术数量（仅"准备施法者"）

博德之门3中有两类施法者：

| 类型 | 职业 | 机制 |
|------|------|------|
| **准备施法者** | 法师、牧师、德鲁伊、圣武士、游侠 | 每天长休后从法术书中"准备"一部分法术 |
| **已知法术者** | 术士、吟游诗人、邪术师、奥法骑士、诡术师 | 法术固定，升级时选择新法术 |

**准备施法者的每日准备数量：**
```
可准备法术数 = 职业等级 + 施法关键属性调整值
```

**示例：** 5级牧师，感知18（调整值+4）
- 可准备法术 = 5 + 4 = 9个法术
- 如果感知只有14（调整值+2），只能准备 5 + 2 = 7个法术

> 💡 这意味着感知高的牧师每天有更多战术选择，可以根据当天预期遭遇准备针对性法术。

### 📊 施法关键属性对战斗的实际影响

假设5级角色（熟练加值+3），面对AC 15、豁免+2的敌人：

| 施法关键属性 | 调整值 | 法术攻击加值 | 命中成功率 | 法术豁免DC | 敌人抵抗成功率 |
|-------------|--------|-------------|-----------|-----------|---------------|
| 14 | +2 | +5 | 55% | 13 | 50% |
| 16 | +3 | +6 | 60% | 14 | 45% |
| 18 | +4 | +7 | 65% | 15 | 40% |
| 20 | +5 | +8 | 70% | 16 | 35% |

**关键洞察：**
- 施法关键属性从14提升到20，**命中率提升15%**，**敌人抵抗成功率降低15%**
- 这相当于全程给你所有法术提供了一个"小优势"！

### 🎯 新手施法者属性分配建议

**创建角色时（27点购点）：**
1. **施法关键属性优先点到16**（通过种族加值可达17）
2. **体质至少14**（维持专注豁免、增加血量）
3. **敏捷12-14**（先攻和AC）

**升级时属性提升优先级：**
- **4级**：施法关键属性 +2（16→18）
- **8级**：施法关键属性 +2（18→20）
- **12级**：体质 +2（14→16）或选择强力专长

> ⚠️ **常见新手错误**：把点数分散投资，施法关键属性只有14。这会让你的法术很难命中、很容易被抵抗，施法体验极差。

### 🔄 兼职时的施法关键属性

如果你兼职多个施法职业，**每个职业使用自己的施法关键属性**，互不影响：

**示例：** 5级法师（智力施法）+ 3级牧师（感知施法）
- 施展法师法术时：使用智力调整值计算攻击/DC
- 施展牧师法术时：使用感知调整值计算攻击/DC
- 法术位总数按"施法等级"合并计算

**施法等级计算：**
- 全施法者（法师、牧师、德鲁伊、术士、吟游诗人、邪术师）：等级×1
- 半施法者（圣武士、游侠）：等级×½（向上取整）
- 三分之一施法者（奥法骑士、诡术师）：等级×⅓（向上取整）

**示例：** 6级圣武士 + 6级术士
- 圣武士施法等级 = 6 × ½ = 3
- 术士施法等级 = 6 × 1 = 6
- 总施法等级 = 3 + 6 = 9级
- 拥有1环×4、2环×3、3环×3、4环×3、5环×1 法术位

> 💡 兼职时虽然施法关键属性分开计算，但**法术位是共享的**。你可以用高等级法术位施展低等级法术（如用3环位施展1环魔法飞弹，伤害更高）。

### 3.3 护甲等级 (AC)

AC 代表敌人命中你的难度。AC越高越好。

| 护甲类型 | AC计算 |
|----------|--------|
| 轻甲 | 基础AC + 敏捷调整值 |
| 中甲 | 基础AC + 敏捷调整值（上限+2） |
| 重甲 | 固定基础AC（不受敏捷影响） |
| 无甲（武僧/野蛮人） | 10 + 敏捷调整值 + 其他加值 |
| 盾牌 | +2 AC |

### 3.4 伤害计算

命中后计算伤害：
```
伤害 = 武器/法术伤害骰 + 属性调整值 + 其他加值
```

**重要概念：**
- **暴击 (Critical Hit)**：D20投出20时自动命中，且伤害骰翻倍
- **大失败 (Critical Miss)**：D20投出1时自动未命中
- **抗性 (Resistance)**：受到某类型伤害减半
- **易伤 (Vulnerability)**：受到某类型伤害翻倍

## 四、优势与劣势

### 4.1 核心机制

这是D&D 5E最独特的机制之一：

- **优势 (Advantage)**：投2个D20，取**较高**的结果
- **劣势 (Disadvantage)**：投2个D20，取**较低**的结果

> ⚠️ **关键规则**：如果你同时有优势和劣势（无论各有多少个），它们**全部互相抵消**，只投1个D20。

### 4.2 优势对命中率的影响

| 基础成功率 | 有优势时 | 有劣势时 |
|------------|----------|----------|
| 40% | 64% | 16% |
| 50% | 75% | 25% |
| 60% | 84% | 36% |
| 75% | 93.75% | 56.25% |

**结论**：优势极其重要！尽量给自己造优，给敌人造劣。

### 4.3 常见获得优势的方法

| 情况 | 效果 |
|------|------|
| 从高处攻击低处敌人 | 远程攻击优势 |
| 敌人处于**倒地**状态 | 近战攻击优势（远程劣势） |
| 敌人处于**目盲/麻痹/昏迷** | 攻击优势 |
| 敌人处于**受惊**状态 | 攻击优势 |
| 影心的引导神力-分身 | 光环内盟友攻击优势 |
| 某些法术（如祝福术） | 攻击检定+1d4 |

### 4.4 常见导致劣势的情况

| 情况 | 效果 |
|------|------|
| 从低处攻击高处敌人 | 远程攻击劣势 |
| 与敌人贴脸使用远程武器 | 远程攻击劣势 |
| 攻击处于重度遮蔽中的敌人 | 攻击劣势 |
| 自己处于**目盲**状态 | 攻击劣势 |
| 超出远程武器最大射程 | 攻击劣势 |

## 五、法术系统

### 5.1 法术位与施法

- 法术需要消耗**法术位**才能施展
- 法术位按等级划分（1环、2环、3环...）
- 长休后恢复所有法术位
- 戏法 (Cantrip) 不消耗法术位，可无限使用

### 5.2 两类法术命中机制

博德之门3中，有害法术分为两大类，命中机制完全不同：

#### 第一类：攻击掷骰型法术

- 像射箭一样，需要施法者进行**攻击检定**
- 法术页面有**准心图标** 🔫
- 命中率 vs 目标AC
- 施法者投D20，加上法术攻击调整值

**公式：**
```
法术攻击检定 = D20 + 法术熟练加值 + 对应属性调整值
```

#### 第二类：豁免检定型法术

- 像出题一样，施法者设定难度，敌人来解题
- 法术页面有**对应豁免属性图标**
- 命中率 vs 敌人豁免能力
- **施法者不投骰**，取固定DC值；**敌人投D20**进行豁免

**公式：**
```
法术DC = 8 + 法术熟练加值 + 对应属性调整值

敌人豁免 = D20 + 敌人对应属性调整值 + 敌人熟练加值（如果有）

如果 敌人豁免 ≥ 法术DC → 抵抗成功/伤害减半
如果 敌人豁免 < 法术DC → 抵抗失败/全额伤害
```

### 5.3 如何选择法术类型？

| 情况 | 推荐 |
|------|------|
| 敌人AC很高 | 使用豁免型法术 |
| 敌人某项豁免很低 | 使用对应属性的豁免型法术 |
| 你能获得攻击优势 | 使用攻击掷骰型法术 |
| 需要稳定输出 | 豁免型法术更稳定（不受AC影响） |

### 5.4 专注 (Concentration)

某些法术需要**维持专注**才能持续生效。同一时间只能维持**一个**专注法术。

**会打断专注的情况：**
- 施展另一个需要专注的法术
- 承受伤害时需要进行**体质豁免**（DC = 10 或 伤害值的一半，取高）
- 陷入乏力或死亡

**常见需要专注的法术：**
- 祝福术、庇护术、魅惑人类、蛛网术、云雾术等

### 6.3 种族选择要点

- **属性加值**是否匹配你的主属性
- **黑暗视觉** —— 精灵、半精灵、提夫林、半兽人等有黑暗视觉，人类和半身人没有
- **特殊能力** —— 如精灵的精灵血统（魅惑免疫）、半兽人的 Savage Attacks 等

**新手推荐种族：**

- **木精灵**：敏捷+2，感知+1，黑暗视觉，移速快
- **黄金矮人**：体质+2，感知+1，黑暗视觉，血量上限+1/级
- **人类**：全属性+1，简单直接

### 6.4 子职业与专长

- **3级**时选择子职业，决定你的专精方向
- **4级、8级、12级**时获得**专长 (Feat)**，可选择属性+2 或 特殊能力

**新手强力专长：**
- **幸运 (Lucky)**：每天3次重投D20，最强通用专长
- **观察力 (Observant)**：感知+1，被动察觉+5
- **属性提升**：简单粗暴，主属性+2

**进阶专长（特定流派）：**
- **巨武器大师**：-5命中 +10伤害，适合高力量战士
- **神射手**：-5命中 +10伤害，适合远程职业
- **战争施法者**：专注豁免优势，适合施法者

## 八、新手实用技巧

### 8.1 战斗技巧

1. **高低差很重要** —— 远程职业尽量站在高处，获得攻击优势
2. **先攻决定顺序** —— 敏捷高的角色先行动，优先解决威胁
3. **集火优先** —— 先消灭一个敌人比打伤多个敌人更有效
4. **控制比输出重要** —— 击倒、魅惑、恐惧等控场技能价值极高
5. **善用环境** —— 推下悬崖、引爆油桶、高处坠落都能造成伤害
6. **注意光照** —— 人类/半身人没有黑暗视觉，昏暗环境下攻击劣势

### 8.2 资源管理

1. **长休 vs 短休**
   - **短休**：恢复部分职业能力，不消耗营地补给
   - **长休**：恢复所有生命值、法术位，但需要消耗营地补给
2. **法术位规划** —— 不要在小怪身上浪费高环法术
3. **药水储备** —— 保持足够的治疗药水和状态恢复药水

### 8.3 探索技巧

1. **频繁按Alt键** —— 高亮显示可交互物品
2. **察觉检定** —— 让感知高的角色走在前面，自动发现陷阱和隐藏物品
3. **保存习惯** —— 进入新区域、战斗前、重要对话前都保存
4. **偷窃技巧** —— 让队友与NPC对话吸引注意，游荡者潜行偷窃

### 8.4 造优去劣 checklist

战斗前检查：
- [ ] 远程职业是否在高处？
- [ ] 是否有敌人可以被推下悬崖？
- [ ] 是否可以使用控场法术限制敌人？
- [ ] 是否给主力输出上了祝福术？
- [ ] 敌人是否有抗性/易伤需要注意？

## 九、进阶构筑思路

### 9.1 兼职 (Multiclassing)

博德之门3允许兼职，最多12级可以分配到多个职业。

**常见强力兼职：**

| 构筑 | 职业分配 | 核心思路 |
|------|----------|----------|
| **索帕** | 5圣武士 + 5邪术师 + 2战士 | 圣武士的至圣斩 + 邪术师的魔能爆 + 战士的二打 |
| **11战1牧** | 11战士 + 1战争牧师 | 11级战士三打 + 战争牧师额外攻击 = 一回合四打 |
| **双持贼** | 5游荡者 + 5战士 + 2其他 | 游荡者的偷袭 + 战士的二打 + 额外动作 |

**兼职注意事项：**
- 需要满足新职业的属性要求
- 施法者的法术位按总施法等级计算
- 某些强力能力在特定等级获得，规划好升级路线

### 9.2 属性点分配建议

**近战输出（战士/野蛮人/圣武士）：**
- 力量 16-17，体质 14-16，敏捷 10-12

**远程/敏捷输出（游荡者/游侠/武僧）：**
- 敏捷 16-17，体质 14，其他按需

**施法者：**
- 主施法属性 16-17，体质 14（维持专注），敏捷 12-14（先攻和AC）

### 9.3 队伍搭配建议

**平衡型队伍（推荐新手）：**
- 前排坦克：战士/圣武士/野蛮人
- 治疗辅助：牧师/德鲁伊
- 远程输出：游侠/法师/邪术师
- 多功能位：游荡者（开锁/偷袭）/吟游诗人（辅助/对话）

**核心原则：**
- 至少有一个治疗者
- 至少有一个高AC前排
- 覆盖多种伤害类型（物理/火焰/闪电/毒素等）
- 覆盖多种技能（开锁、察觉、说服等）

---

## 总结

博德之门3的规则看似复杂，但核心逻辑非常清晰：

1. **D20 + 修正值 vs 目标值** —— 所有检定的统一公式
2. **优势 = 投2取高，劣势 = 投2取低** —— 战斗中的核心策略
3. **动作经济** —— 每回合的动作、额外动作、反应都要充分利用
4. **资源管理** —— 法术位、药水、长休/短休的合理规划

掌握这些核心概念后，你就能在费伦大陆上畅通无阻了！祝你冒险愉快！⚔️🎲
